# CaterCats Website

A whimsical, cat-themed catering service website featuring fine dining with feline flair.

![CaterCats Website Preview](Images/screenshot-1762012678787.png)

## Description

CaterCats is a fun and elegant website showcasing a fictional cat-operated catering business. The site includes information about the team of cat chefs and staff, menu highlights, customer reviews, and booking information.

## Features

- **About Section**: Introduction to CaterCats catering services
- **Team Section**: Meet the CEO and two catering teams with their feline staff members
- **Menu Highlights**: Showcase of signature dishes
- **Reviews**: Customer testimonials
- **Booking Section**: Contact information and booking button

## Technologies

- HTML5
- CSS3
- Responsive design with media queries

## Files

- `Catercats.html` - Main HTML file
- `Catercats.css` - Stylesheet
- `Images/` - Directory containing team member photos and menu item images

## Usage

Simply open `Catercats.html` in your web browser to view the website.
